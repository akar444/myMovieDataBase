# myMovieDataBase

### Installation

```
1. Clone the project in a directory using command -

        git clone https://github.com/akar444/myMovieDataBase.git

2. After cloning the project navigate to directory myMovieDataBase
3. Run command
        npm install
   to install all node dependencies.
4. Run command
        bower install
   to install all client side libraries like angular,bootstrap and jquery
5. Now run command
        npm start
   to start the server.It will start the server at
        http://localhost:8000
   Open this in browser it will show all movie list.

```
